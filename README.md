# VUE & NESTJS MONOREPO TEMPLATE

## Purpose
```
To create a template file for future Vue and NestJS projects under a monorepo project 
```

### Description
```
Uses Vue as the Frontend, NestJs as the Backend, DB is up to the requirement of the project, may go with Postgres

```
### Instructions

```
1. Initialize Git on file
  
  - Once received the full project, initialize git on it to set all the files/projects under it to be under one monorepo

  - Refer to link: https://kbroman.org/github_tutorial/pages/init.html


2. Start working on your new Project!

  - May need to create docker files to setup containers for the individual projects, DB and a docker-compose.yaml file on the main project file
```
